# B4 · Template Dashboard · 模板数据看板

> 70+ 爆款 Hook 模板的统计端 · 让 Coach 知道每个模板「最近数据怎么样、需不需要淘汰、谁拍得好」

## 🎯 The Problem

70+ 爆款 Hook 模板沉淀下来后，Coach 团队需要持续判断：
- 哪些模板还在跑（最近数据好）
- 哪些已经废了（命中率断崖式下跌）
- 哪些大使拍这个模板最有效（适合定向派活）

凭经验判断容易遗漏，需要看板支持。

## 💡 The Approach

模板维度的数据看板：

- **每个模板的近期数据**：播放 / 互动率 / 爆款率
- **模板淘汰判断**：长期低于阈值的标红
- **谁拍这个模板效果好**：按模板 × Creator 的命中率排序
- **跨账号 × 跨平台透视**：同一个模板在 TikTok / IG / YT 三平台的表现对比
- **必拍模板覆盖度**：每位 Creator 必拍模板的完成情况

## 📊 Key Results

- ✅ 配合 Coach 给 Creator 定向派活，**提高模板命中率**
- ✅ 模板淘汰决策从"凭感觉"→"看数据"
- ✅ 反哺 [D1 视频效果分析](../../04-data-analysis-insights/D1-video-effectiveness/) 的模板维度切片
