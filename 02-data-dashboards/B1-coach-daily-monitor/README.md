# B1 · Coach Daily Monitor · 达人每日表现实时大屏

> 管理层和 Coach 每天上班第一件事就打开它 · 决策周期从 2 周缩短到每天

## 🎯 The Problem

原本管理层和 Coach 每天需要：
- 手动拉 Excel 表
- 翻 Slack 群消息
- 凭印象拼出"昨天这帮人到底干了什么"

**决策周期长、数据时效差**，往往等到周末复盘才能看清趋势，等做出调整动作时机会已经过了。

## 💡 The Approach

一个每日表现实时大屏，覆盖多个追踪维度：

| 维度 | 内容 |
|------|------|
| 发布进度 | 当日 / 当周 / 累计视频数 |
| Quota 完成 | 每位 Creator 的任务完成度 |
| 视频表现 | 播放量 · 互动率 · 爆款标记 |
| 模板命中 | 哪些模板被采用 + 表现 |
| 账号风险 | 异常账号（数据骤降 / 投放停止）|
| Coach 待处理事项 | 待审 / 待回复 / 待结算 |

### 后续增强
- **CPM 字段**：支持流量价值评估（流量贵不贵，单位成本回报）
- **跨期数据快照**：便于趋势对比（不只是看当下，能对比上周 / 上上周）
- **按 Coach 维度过滤**：每位 Coach 只看自己下属

## 📊 Key Results

- ✅ 上线后**管理层和 Coach 每天上班都会先打开**看下属在当前时段或前一天的视频拍摄与上传数量、近期 Views 数据，并通过 CPM 同比辅助决策
- ✅ **决策周期从原来的两周缩短至每天**，每周做一轮人员优化
- ✅ 配合 A5 达人管理 + B3 评分模型，形成"监控 → 评估 → 调整"的闭环
