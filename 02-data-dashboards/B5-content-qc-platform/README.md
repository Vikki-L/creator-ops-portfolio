# B5 · Content QC Platform · 社媒 QC 平台

> 把各业务模块的内容质量管控（QC）需求**整合成一张总览图** · 三分结构 · 现状盘点 + 长期规划

## 🎯 The Problem

各业务模块的 QC（质量管控）需求分散：
- A1 视频审核是一个 QC 入口
- A3 AI 审核是另一个 QC 入口
- B1 Daily Monitor 也有 QC 信号
- 内部团队需要一张**总览**回答"我们的内容质量到底怎么样"

但缺乏一个统一的视角和长期规划。

## 💡 The Approach

设计了**「三分结构」**总览：

- **Creator 平台**（A 板块为主）
- **Competitor 平台**（B2 为主）
- **Infra 总览**（数据流转 + 看板入口）

输出：
- **现状盘点**：把当前各 QC 入口的责任 / 数据源 / 状态列清楚
- **长期规划**：未来 6-12 个月的演进路径

## 📊 Key Results

- ✅ 形成 **QC 长期规划文档**（团队对齐用）
- ✅ **现状总览原型**（让管理层一眼看清 QC 资产分布）
