# D1 · Video Effectiveness · 视频效果分析

> 周期性视频内容数据分析 · 拆解什么样的视频最有效 · 反哺 Template 库 + Coach 派活策略

## 🎯 What

把过去一段时间的视频按**内容类型**（Trend / Talking）和**模板**分类，分析：
- 哪类内容更容易出爆款
- 哪类模板命中率最高
- 各平台（TT / IG / YT）的差异表现

## 💡 Approach

- **Trend vs Talking 内容分类**与表现分析
- **模板匹配分析**（template match）
- **爆款视频拆解**（5K+ rate / 10K+ rate）
- **周 / 双周维度**分析报告
- **按周按平台转化整理**：跨周横向对比

## 📊 Outcomes

- ✅ 持续反哺 [B4 Template 数据看板](../../02-data-dashboards/B4-template-dashboard/) 的模板迭代
- ✅ 反哺 Coach 给 Creator 的拍摄指导（哪种 Hook 这周最有效）

## 🔧 What I Owned

- 分析框架设计（Trend vs Talking 分类法 + 爆款率口径）
- 周期性报告输出
- 跨周横向对比汇总表

## 📂 Related (Internal)

- 多份周/双周分析 Excel 报告
- 社媒按周按平台转化整理表
