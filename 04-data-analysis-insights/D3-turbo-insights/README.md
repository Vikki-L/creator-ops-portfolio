# D3 · Turbo.ai Insights · 「三段踢人」方法论

> ⭐ 这不是单纯的"竞品研究" —— 这是一个完整的「**数据洞察 → 方法论 → 产品落地 → 业务决策**」闭环。

## 🎯 Core Insight

> **一位 Creator 拍到 ~30 个视频左右，就能稳定判断他/她会不会出爆款。**

这个洞察来自对 Turbo.ai 几百位 Creator 完整视频数据的拉取与回归分析——观察他们从入职到爆款的视频累计数分布，发现"出爆款"的能力在 **~30 视频前后会显著分化**。

## 💡 Methodology · 「三段踢人」标准

基于上述洞察，提炼出一套**按活跃度 + 数据表现的 3 段判定标准**，辅助 Coach 团队做清退决策：

| 段位 | 判定信号 | 推荐动作 |
|------|---------|---------|
| 第一段 | （内部文档）| 重点培养 |
| 第二段 | （内部文档）| 观察 / 警告 |
| 第三段 | （内部文档）| 清退候选 |

## 🔗 Landing Loop · 落地链路（这个项目的真正价值）

```
1. 拉取 Turbo.ai 几百位 Creator 完整视频数据
   ↓
2. 回归分析发现 "30 视频判断爆款" 的拐点
   ↓
3. 形成「三段踢人」方法论
   ↓
4. 落地到 B3 校园大使潜力评分模型
   （max views / median / 5K 突破率 / 10K 突破率 / 100K 爆款率 / 早期突破 / 阶段判断）
   ↓
5. 输出 4 级分级 + 推荐动作（HIGH / MEDIUM / LOW / DROP RISK）
   ↓
6. 基于学期数据辅助 Coach 团队清退约 100 位大使
   ↓
7. 配合 A5 Coach 达人管理流程，达人池整体质量明显提升
```
