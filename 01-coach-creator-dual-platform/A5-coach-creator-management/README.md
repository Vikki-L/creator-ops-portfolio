# A5 · Coach × Creator Management · 内部 CRM · 达人池管理

> 把散落在多个 Excel / 飞书表的达人信息整合成内部 CRM · 200+ 大使统一管理

## 🎯 The Problem

达人信息散落在多个 Excel / 飞书表里：
- 招募时填的表
- Coach 自己维护的小组表
- 状态变更（清退 / 激活）没有统一记录
- Coach 团队无法跨表汇总，无法统一管理 200+ 位达人的状态

## 💡 The Approach

把达人池管理做成内部 CRM 子系统：

- **Coach 与内部人员表关联设计**（达人归属哪位 Coach 一目了然）
- **达人池管理流程**：清退 / 激活 / 跟进 三态流转
- **活跃度自动化清退建议**（数据驱动，结合 B3 评分模型）

## 📊 Key Results

- ✅ 当前 Active 大使 **125 位**（持续清退优化的结果）
- ✅ Coach 不再依赖手动 Excel 维护，操作直接在系统里完成
