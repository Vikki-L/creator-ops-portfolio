# A6 · Creator Earnings · 大使薪酬展示（Creator 端）

> Creator 端的薪酬可视化页面 · 与 A2（Coach 端结算）双端联动 · 自助确认入口

## 🎯 The Problem

Creator 端原本看不到自己当前的：
- 本周分级（Tier）
- 本周收入预估
- 每条视频对应的薪酬 + Bonus
- 历史结算记录

→ 全靠在 Slack 上私聊 Coach 一次次问 → Coach 重复劳动 + Creator 信息黑盒。

## 💡 The Approach

Creator 端 Earnings 页面：

- **本周分级显示**
- **视频明细**（每条视频对应的薪酬 + Bonus）
- **累计 / 历史结算记录**
- **自助确认入口**（与 A2 联动，Creator 在自己账号确认薪酬明细）

## 📊 Key Results

- ✅ 减少 Coach 对个体咨询的接待量
- ✅ Creator 对薪酬可视化 → 投入度提升
- ✅ 自助确认机制减少结算争议
