# 03 · AI Workflow Automation
## AI 工作流自动化

> 用 Bot 接管 Coach 和 Onboarding 团队的重复劳动，让人专注做策略和创意。

## Overview

两个不同平台、不同对象、不同场景的 Bot：

| Bot | 平台 | 对象 | 接管什么 |
|-----|------|------|---------|
| **Onboarding Bot** | Discord | 新加入的 Creator | 5 步入职自动化 + FAQ 智能回答 |
| **Coach Assistant Bot** | Slack | Coach | 多大重复工作：催回复 / 邮件升级 / 视频初审 / 模板格式提醒 / 5K+ 评论奖励 / 问答 |

## Modules

| 模块 | 一句话 | 状态 |
|------|--------|------|
| [C1 · Onboarding Bot](./C1-solvifox-onboarding-bot/) | Discord 新人入职自动化（5 步） · 含完整 Python 数据 pipeline 作为二期优化依据 | ✅ 已落地 |
| [C2 · Coach Assistant Bot](./C2-coach-assistant-bot/) | Slack 上替 Coach 做催回复 / 初审 / 模板提醒 / 5K+ 评论奖励（多功能）| ✅ 已规划 |
