## 📌 About

这是一个北美 AI 教育出海产品的内部产品系统建设作品集。我从 0 到 1 主导设计了一套覆盖**校园大使全生命周期**的协作产品 + 数据看板 + AI 工作流，把原本散落在 Slack / Discord / 飞书的运营链路整合成统一的内部系统。

每周覆盖：
- 🎬 **2000+ 视频** 在 TikTok / Instagram / YouTube 三平台投放
- 👥 **200+ 北美校园大使** 内容生产
- 🧑‍🏫 **10 人内部 Coach 团队** 协作
- 💸 **跨境薪酬结算** 通过美区薪酬平台自动化打款
